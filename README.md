#  Amazon Prime Movies and TV Shows - EDA Project

##  Overview
This project performs **Exploratory Data Analysis (EDA)** on Amazon Prime content, including movies and TV shows. The aim is to extract meaningful insights from the dataset and understand trends in content distribution, ratings, and production.

---

##  Dataset
The dataset consists of two files:

- **titles.csv.zip** → Contains details about movies and TV shows  
- **credits.csv.zip** → Contains information about actors and directors  

###  Features:
- Title, Type (Movie / TV Show)
- Genre, Release Year, Runtime
- Age Certification
- IMDb & TMDb Ratings
- Production Countries
- Cast & Crew Details

---

##  Data Cleaning
- Removed missing and duplicate values  
- Filled null values using mean / default values  
- Converted data types properly  
- Cleaned and formatted columns  
- Handled list-type columns (genres, countries)  

---

##  Exploratory Data Analysis

###  Key Visualizations:
- Movies vs TV Shows distribution  
- Top genres  
- Content trend over years  
- IMDb score distribution  
- Runtime analysis  
- IMDb vs TMDb correlation  
- Top actors and directors  
- Country-wise content distribution  

---

## Key Insights
-  Most content is **Movies**, fewer TV Shows  
- **USA dominates** production, followed by India & UK  
-  IMDb scores mostly lie between **6–7**  
-  Strong correlation between IMDb and TMDb ratings  
- Certain directors contribute large number of titles  

---

##  Business Recommendations
- Increase content from **diverse countries**  
- Focus on **high-rated genres** like drama & documentaries  
- Add more **family-friendly content**  
- Invest in **quality over quantity**
